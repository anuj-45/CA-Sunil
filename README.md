# CA Professional Services Website

A modern, responsive website for a Chartered Accountant (CA) firm showcasing professional accounting services.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Key Sections**:
  - Hero section with call-to-action buttons
  - About section with statistics
  - Services showcase (6 core services)
  - Contact form with contact information
  - Professional footer with quick links

## Services Highlighted

1. Tax Planning & Filing
2. Auditing Services
3. Business Consulting
4. Financial Planning
5. Company Formation
6. GST Services

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (no frameworks)

## How to Use

1. Open `index.html` in your web browser
2. Navigate through sections using the navigation menu
3. The website is fully functional with smooth scrolling and interactive elements
4. Mobile-friendly with hamburger menu for smaller screens

## Customization

To customize this website for your CA firm:

1. **Update Contact Information**: Edit the contact details in the Contact section
2. **Modify Services**: Update the services in the Services section to match your offerings
3. **Change Colors**: Modify CSS variables in `styles.css`:
   - `--primary-color`: Main brand color
   - `--secondary-color`: Accent color
   - `--accent-color`: Highlight color
4. **Add Logo**: Replace the text logo in the navbar with your company logo
5. **Form Integration**: Connect the contact form to your backend or email service
